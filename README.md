**C Programming Practice – The C Programming Language (K&R)**

This repository contains my solutions to exercises from the classic programming book “The C Programming Language” by Brian W. Kernighan and Dennis M. Ritchie.
The goal of this repository is to strengthen my understanding of C programming fundamentals by implementing solutions to the exercises provided in the book.

**📚 About the Book**

The C Programming Language (often referred to as K&R) is one of the most influential programming books ever written. It introduces the C language with clear explanations and practical exercises.

Through this repository, I am working through the book chapter by chapter and solving the exercises to deepen my understanding of:
C syntax and semantics
Memory management
Pointers
Data structures in C
Low-level programming concepts

**🎯 Goals**

Master core C programming concepts
Improve problem-solving skills
Gain a deeper understanding of pointers and memory
Build strong foundations for systems programming

**💻 Language**

All solutions are implemented in:
**C**

**⚙️ Compilation & Execution**

You can compile and run the programs using gcc:
`gcc program.c -o program`
`./program`

Or with warnings enabled:
`gcc -Wall -Wextra program.c -o program`
`./program`
